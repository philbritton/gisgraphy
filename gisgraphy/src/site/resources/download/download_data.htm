download server

If you are interesting in downloading data you can go to :

  • The Gisgraphy download server. You will find
    • The raw Openstreetmap PBF files splited by country
    • All the streets, cities, Point of interests, and house numbers from Openstreetmap and converted in CSV format to facilitate the import. There is one file per country and a file with all the countries. In addition to the native openstreetmap data, some pre-calculated fields (length, middle point) are added to the csv format.
    • The shape of more than 160 000 cities and localities extracted from Quatroshapes with their associated geonames Id


    Gisgraphy process pipeline (click to enlarge).

  • The Geonames download server if you need to download places

You DON'T have to download data by yourself, to import tham into Gisgraphy : The gisgraphy importer will download it for you. You only need to download it if you are interested to use the data in an other way that Gisgraphy

Custom / premium dump

If you need some dumps with fields that are presents in Openstreetmap but not in the csv files (e.g : speed limit) or if you need more recent dump, it is possible to order custom dump
