Benchs

The benchs have been made :
  • on a core2duo 2Ghz, and gisgraphy has been launch with 2G
  • The bench dataset have been extracted from the database, and all the data are unique to minimize cache usage
  • before each bench I restart the server
  • The benchs scripts are availables here
