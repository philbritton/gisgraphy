osm2gisgraphycsv


A project called osm2gisgraphycsv has been created by willy tiengo and marcio aguiar to convert the Openstreetmap data from the openstreetmap siteopenstreetmap site to the Gisgraphy CSV format.

It gives the ability to add some files to the import. This program is not used by Gisgraphy to compute the files in the download server but is an external tool. It is not develop by me, and no support will be given.
