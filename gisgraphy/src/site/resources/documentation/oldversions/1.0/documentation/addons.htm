Gisgraphy addons

Gisgraphy is available in multiple adons :
  • Fulltext Webservices and Fulltext Webservices
  • Fulltext Java API and Geoloc Java API
  • OpenSearch Module
  • Ajax / javascript API

OpenSearch Module

OpenSearch is a collection of simple formats for the sharing of search results.
It is supported by Internet Explorer (for versions >= 7) and Firefox (for versions >= 2).
It give the possibility for users to add search engine in the search box of the browser :
