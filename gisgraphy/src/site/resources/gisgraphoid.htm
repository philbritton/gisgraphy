About

Gisgraphoid is a part of the free opensource gisgraphy geocoder (as the Gisgraphy address parser, and the Gisgraphy geocoder), it gives you the ability to do geocoding on your Android mobile or tab with the same Geocoder API as Google but with Gisgraphy. Free, No Google API needed by the phone, no API key, no limit. You can display the result on Openstreetmap or Google maps (Google API key required). Gisgraphy geocoder can be used when no geocoder is present.

Gisgraphoid is package as
  • A jar
  • An apklib

Both can be downloaded at Gisgraphoid Google code page
Gisgraphoid :
  • contains The Gisgraphy geocoder (of course :) )
  • contains a mocked geocoder to simplify development
  • Use Maven
  • have some useful layout (e.g : a layout to display results on an Openstreetmap map with OSMDroid)
  • all the tests resources. that allow to modify the code and rerun the tests to avoid regressions

Demo

A demo application called "gisgraphoid demo" give you some example of code to use Gisgraphoid. You can find it on the Android market or download it on Google code. All is free and open source (Browse sources).

Terms and conditions

Gisgraphy goal is to provide a free opensource and production ready geocoder. You can test all the Gisgraphy webservices on services.gisgraphy.com. You can find all the necessary informations on Terms and conditions on that page. You can use gisgraphoid with the free webservices for testing but don't use the free services for production, you can easily install Gisgraphy on your own server or if you need we can provide premium services. For the moment Free services does not require Gisgraphy API key but if people abuse, i will be forced to restrict the access :(
