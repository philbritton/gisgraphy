Leaflet plugins

2 Leaflet plugins are availables. Geocoding plugin and Reverse geocoding plugin
Try the demo with the two plugins

Geocoding plugin

Try the leaflet geocoding demo

This plugin allows to add a geocoding input with autocompletion on a leaflet map. It wraps the Gisgraphy js API. It manage :
  • A place : paris, big apple
  • An address : Avenue des Champs-Élysées Paris
  • A GPS : 48.873409271240234,2.29619002342224
  • A DMS : 40:26:46.302N 079:56:55.903W
  • A magic phrase : restaurant new york




The plugin is highly configurable, and a lot of options are available. The documentation does not give all the options and possiblilites, but we strongly recommend to have a quick look at the code.

Reverse geocoding plugin

Try the leaflet reverse geocoding demo
This plugin simply put an HTML popup when you rightclick on the map. It is a sample and you can personalize it change the right click to the left one, ...
