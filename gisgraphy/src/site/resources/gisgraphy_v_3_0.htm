Gisgraphy V 3.0 is out !

To folow the one version / per year (and After more than one year of development), I am proud to announce that the Version 3.0 of Gisgraphy is now available. It is a major versions where a LOT of functionalities has been added :
  • Importer have been improved and take less time to run (about 25%)
  • Dedicated Geocoding webservice.
  • Address Parser web service.
  • Real fulltext search for street via SolR.
  • Streets are linked to their city.
  • Importer wizard.
  • Possibility to reset the import automatically via admin interface.
  • Possibility to specify a GPS point for fulltext search.
  • Zipcode importer.
  • Migrate to a newer SolR version.
  • More output format supported.
  • Extract some module and librairy
  • Possibility to consider that all words as not required in a fulltext search.
  • User interface improved.
  • Admin interface improved.
  • CRUD interface to edit database entry(streets and place)
  • Gisgraphoid : Geocode on Android with same API as Google but with Gisgraphy
  • ....Read the complete change log
