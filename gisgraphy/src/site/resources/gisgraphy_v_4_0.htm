Gisgraphy V 4.0 is out !

It has been longuer than expected, but, I am proud to announce that the Version 4.0 of Gisgraphy is now available. It is a major versions where a LOT of functionalities has been added :
  • A new dedicated server has been added
  • A new (MONSTER) server is now dedicated for import
  • 100% of data has been added!
  • Openstreetmap Pois are now imported
  • Openstreetmap cities with their shapes are now imported
  • possibilty to restrict the import to a given list of place types
  • A lot of place types added (Cinema, Dentist, Doctor, Fuel, NightClub , Pharmacy, Rental, Shop, SwimmingPool, Taxi, Telephone, Toilet, Veterinary,...see all)
  • You can now download raw PBF files splited by countries on the download server
  • Importer now download all the files it needs before starting (fail fast if all files are not there)
  • House numbers are now imported
  • Street has now alternate names
  • Dedicated reverse Geocoding webservice.
  • countrycode is no longuer required for geocoding ! :)
  • Address parser has been STRONGLY improved and is now a dedicated product
  • We can now makes difference in Geonames between populated place and cities(geonames has no cooncept of cities just populated place that can be suburb, quater , whatever)
  • Possibility to geocode structured addresses
  • Relevance has been improved
  • Better management of alphabet, accent and special char (e.g :oe)
  • Some libraries has been extracted to makes modules
  • Geocoding and fulltext service now return zip (is_in_zip), adm (is_in_adm), city(is_in_city), and place (is_in place)
  • Fulltext engine now return the scoring
  • We remove duplicates alternatenames and zip codes between Geonames and openstreetmap
  • Migrate to a newer SolR version >3.6.
  • User interface improved.
  • Admin interface improved.
  • documentation updated
  • a lot of bugs have been fixed
  • feedbascks have been taking into acount
  • ....Read the complete change log and all commits



You can download it and give your feedbacks. You can also try it on the free server.
