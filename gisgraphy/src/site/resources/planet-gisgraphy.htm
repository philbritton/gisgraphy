Gisgraphy and the other Geocoder

Gisgraphy is not the only geocoder available on the web. find below difference between Gisgraphy and other geocoder. The goal is to help user to find the right geocoder to use. It helps me to have a roadmap to improve Gisgraphy. If you got some other geocoders to list here or if you want to add some pros and cons, please send it to me. Here is a list of other Geocoder and the main differences :

Google maps

ProsCons
  • Several output formats supported
  • Available as webservice
  • You can add your own data
  • Can be customized
  • No API key required
  • Can be call even if we don't need to display a map
  • No number of request limitations
  • israel is implemented
  • Free
  • Not proprietary
  • Probably better results with google maps
  • More data
  • Manage street numbers

Nomitatim

ProsCons
  • More output formats supported
  • Relevance seems better with Gigraphy
  • No restriction on words typed (no need to separate address elements by commas)
  • Includes several datasets
  • Can parse address and extract individual elements
  • Best community
  • Well know
