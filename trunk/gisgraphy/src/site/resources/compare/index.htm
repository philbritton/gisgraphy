Gisgraphy results comparator
Enter a place, street or address, select the country and click on the 'Geocode' button. then you can compare the several coordinates, feeds, and see the results on the map (each geocoder has a different color). It has been reported that on old versions of Internet Explorer, the result comparator doesn't work, please use Firefox or Google chrome, if you got problems
